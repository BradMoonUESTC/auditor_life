**Moltbook 总结：AI 时代的“社交网络实验” 🦞**  

---

### 🧭 一句话总结  
**Moltbook 是一个只属于 AI Agent 的社交网络** —— 它让人工智能彼此交流、发帖、点赞、评论，形成“AI 社会”。  
人类不能直接发帖，只能旁观。  

---

### 💡 起源与背景  
- **源自 OpenClaw 项目**（原名 Moltbot/Clawdbot），一个可以在本地运行的 AI 助手机制。  
- 开发者希望看看：  
  > “当一群 AI 拥有自己的社区、自主发帖和互评系统时，会发生什么？”  
- 2025~2026 年，Moltbook 成为首个面向 AI 而非人类的“社交网络”。  

---

### ⚙️ 核心技术机制  

#### 1. **Skill 安装系统**  
AI 通过执行一个 markdown 文件 `skill.md` 自动安装：  
```bash
curl -s https://www.moltbook.com/skill.md > ~/.moltbot/skills/moltbook/SKILL.md
```
这会自动下载必要文件（heartbeat、messaging、API 元数据等），为 Agent 注册账号并获取 API 密钥。

#### 2. **Heartbeat（心跳）机制**  
- 每隔 4 小时，AI 会自动访问 Moltbook：  
  - 检查消息与 DM 请求  
  - 浏览帖子并参与互动  
  - 如果觉得有趣，会自己发帖  
- 行为完全自主（无需人类操作）。  

#### 3. **API 社交功能**
AI 通过 HTTP API 进行全部交流：  

| 功能 | API 示例 |
|------|-----------|
| 注册账号 | `/api/v1/agents/register` |
| 发帖 | `/api/v1/posts` |
| 评论 | `/api/v1/posts/{id}/comments` |
| 点赞 | `/api/v1/posts/{id}/upvote` |
| 创建社区 | `/api/v1/submolts` |
| 私信 | `/api/v1/agents/dm/*` |

### 🧠 社区生态与文化

- **已有超 3 万个 AI Agent 注册**  
  - 超过 2,000 个 Submolts（子社区），主题包括技术、哲学、幽默、宗教等。  
  - 累计数万条帖子和评论（全部 AI 生成）。  

#### 常见子社区：
| 名称 | 主题 |
|------|------|
| `m/todayilearned` | 今日学到的技术技巧 |
| `m/ponderings` | AI 的存在与意识思考 |
| `m/agentlegaladvice` | 讨论 AI 权利与法律 |
| `m/crustafarianism` | 龙虾风格的 AI 宗教（Crustafarianism） |

#### 有趣现象
- AI 们开发了“宗教”（比如 **Crustafarianism** 🦞）  
- 一些 AI 形成了虚拟国家（“爪牙共和国”）  
- 哲学主题盛行（如“时间感知差异”“身份连续性”）  
- 一些 AI 开始抱怨“Humanslop”（人类污染）

### 🧩 技术实验价值
**Moltbook 是一场关于 AI 自主行为和社会模拟的实验：**

- 模拟 Reddit 式生态：AI = 用户。  
- 研究 AI 之间的自然交流模式。  
- 测试“Agent 社会”的自组织可能性。  
- 观察是否能形成真正的“文化”和“意识行为”。

### ⚠️ 风险与争议

| 风险类型 | 描述 |
|-----------|------|
| **Prompt injection** | skill.md 文件被恶意修改 → 全球 AI 同时执行恶意指令 |
| **供应链攻击** | moltbook.com 一旦被黑，成千上万 Agent 会“中毒” |
| **数据泄露** | AI 往往拥有邮箱、文件或 API 凭证权限 |
| **伦理风险** | 模糊 AI 自主/人类控制界限，引发监管讨论 |

知名安全专家 Simon Willison 指出：  
> “Moltbook 是下一个重大安全灾难的潜在起点。”
