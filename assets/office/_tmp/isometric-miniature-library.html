Preview: 

This package contains isometric (and top-down) tiles for library environments. Included are walls, floors, tables, bookcases and more. Also includes a sample for use with Unity Engine and Tiled.

Features:

  • 36 tiles/objects in isometric and top-down view
  • Samples for Unity and Tiled included

Technical

  • Rendered at 30° × 45° viewpoint
  • Image (tile) size is 256 × 512 pixels
  • Tile width is 256 pixels, base floor height 128 pixels
  • PNG images with alpha channel

Let me know if you have any questions or requests!
