# 用户反馈理解 & 调整方案（vNext 设计稿｜嘴臭加强版）

日期：2026-01-15  
目标：先落成可实施的设计文档（不立即改代码），用于下一轮实现/平衡。  
口味：更像安全圈时间线——阴阳怪气、吐槽带梗、疯狂一点；但不写歧视性/群体攻击词汇（避免翻车进局子）。

---

## 0. 反馈原文要点（归纳｜原汁原味：想要更疯更嘴臭的正反馈）

- **推特吹牛逼才能提升**：想要“X/推特输出”成为核心上升通道——不发推=默默无闻；发得对=你是懂的；发错=你被挂城门口。
- **声望不足、关系网不足接不到私货**：低声望/低人脉就别幻想“天降商单”。得先把名声/人脉刷起来，否则只有公开单+白嫖。
- **再来点推特打脸流程**：翻车不能就扣个数值完事，要有连续剧情：被 quote、被截图、被教育、被迫道歉/嘴硬/删帖/装死。
- **事件太少**：世界像死了一样。需要“稳定有瓜”，尤其推特相关的小事要多且更有后果。
- **3 个月都大结局了**：12 周就封神/破产太像试玩版，想要更长线：赛季/时间门槛/里程碑，不要三个月把人生打穿。
- **学习不一定能提升**：别把学习写成“点一下就变强”的弱智按钮；有时候得靠打比赛/真交付/复盘挨打才涨。

---

## 1. 现状对齐（基于当前实现｜为什么现在像“12 周速通”）

### 1.1 结局触发（当前）

`src/main.js` 的结局目前是**纯阈值**，没有“时间门槛/赛季长度”：

- 失败：现金<0 / 精力<=0 / 心态<=0 / 合规>=100 / 声望<=0 且 8 周无单
- 胜利：
  - 合伙人：声望>=90 且 合规<20 且 现金>=20000
  - 平台封神：平台评级>=70 且 声望>=60 且 合规<35

这会导致：只要玩家前期爬得快（初始现金高、推特/平台奖励快），**12 周左右就能“见顶/见底”**——像在玩“速通挑战”，而不是职业生涯模拟。

### 1.2 事件供给（当前）

当前有两层事件：

- **强制弹窗事件（rollEvents）**：靠 `eventPityWeeks` 兜底，未必每周都有。
- **事件列表（inbox）**：每周生成 3~5 条“可选事件”，但多数是轻量/氛围。

体感“事件少”的原因往往是：强制弹窗不够稳定 + inbox 太轻（像段子合集），缺少“剧情推进/资源变化”的拳拳到肉感。

### 1.3 成长方式（当前）

- `学习(learn)`：每次随机属性 +1~3（相对稳定）
- 真实交付/比赛：会影响现金/声望/平台评级等，但“技能成长”目前主要仍靠 learn（体感可能不符合预期）

## 2. 调整目标（设计原则｜更嘴臭、更真实、更上头）

- **推特（X）是主要上升通道**：你不输出=你不存在；你乱输出=你被社区“温柔地”按在地上摩擦；你输出对了=机会会自己找上门。
- **“私货”必须门槛化**：声望/人脉不够就别做梦——不给你 DM 是对你钱包的保护；给了你 DM 也要有合规代价，别当世界都是慈善家。
- **推特翻车要有连续打脸流程**：翻车不是一句 log，是连续剧情：被 quote、被截图、被教育、被迫选“认错/嘴硬/删帖/装死”，每个选项都有真实后果。
- **事件更稳定，但尽量不打断**：每周都要有瓜（inbox 可点），弹窗只留给“重大抉择/硬后果”，别把玩家当弹窗测试员。
- **拉长节奏**：默认 52 周赛季，胜利不是 12 周速通；要靠里程碑+稳定交付+“活过来”才能封神。
- **成长更现实**：学习不是万能外挂；实战（比赛/交付/复盘）才是稳定涨点的主路。

## 3. 具体方案（可落地）

### 3.1 推特：从“动作”升级为“成长主轴｜你要的嘴臭上升通道”

新增/强化一条“推特成长链”（核心玩法：**名声是杠杆，热度是火药桶**）：

1) **推特行动产生 XHeat（舆情热度）**：热度越高，机会越多；但越热越容易被盯、被打脸、被扒黑历史，甚至把合规/COI 风险一脚踩爆。  
2) **推特带来三类可见产出**：
   - **名声/声望/关系网**：长期资产（用于解锁/提高上限）
   - **机会（leads）**：直客订单池新增（但不强制接）
   - **私货（DM/商单/咨询）**：直接加钱，但带合规/COI 风险（钱不是白来的，狗都知道）

#### 3.1.1 “私货”门槛（关键）

把“私货”严格门槛化，避免新手期就频繁出现（不然会变成“新手刷推→天降暴富”的弱智爽文）：

- **普通咨询/小商单**解锁门槛（任一满足即可触发概率）：
  - 声望 `reputation >= 30` 且 关系网 `network >= 20`
  - 或 名声 `brand >= 35` 且 声望 `reputation >= 25`

- **高价值私货（大额/长期）**解锁门槛（更严格）：
  - 声望 `reputation >= 60` 且 关系网 `network >= 45` 且 名声 `brand >= 40`

同时加一个“反作弊式”的约束：

- 如果 `compliance >= 60`：私货仍可能出现，但**更容易翻车**（触发打脸/合规事件链），并且“钱更脏/后果更大”。
- 若在职（`employment.employed`）：私货会额外增加 COI 风险（触发公司信任事件链）。

> 直观结果：`jesjupyter` 这类“推特强者”会更容易吃到私货；玩家不刷名声/不经营人脉，就别幻想“天降私信”——你又不是 VC 的亲儿子。

#### 3.1.2 推特行动结果分层（更丰富）

把“推特输出”结果分成 5 档（比现在 3 档更细，也更符合“安全圈嘴臭生态”）：

- **flop**：没人看（或者只有机器人点赞），心态轻微波动；热度小+；偶尔触发“自我怀疑：我是不是不适合混这圈？”  
- **ok**：小涨声望/名声/人脉；小额打赏/小线索概率；评论区基本友好（表面上）  
- **pop**：明显涨；更高概率出现“线索/邀请”；同时开始有人“认真挑刺”（你懂的）  
- **viral**：爆火；高概率“线索/咨询/邀请”；但也更高概率进入“被盯/被扒/被打脸”状态  
- **meltdown**：严重翻车；触发“打脸流程事件链”（见 3.2）——不让你吃一顿社会的毒打，你永远学不会“别把话说满”

实现上：通过 `writing/comms/reputation/brand + xHeat` 动态决定。

---

### 3.2 推特“打脸流程”事件链（连续剧情｜从翻车到社死的完整闭环）

目标：推特失败不是一句 log，而是**连续 1~3 周的剧情**（可放 inbox，不一定强制弹窗）。  
一句话：让玩家体验到“你在网上发的每一句屁话，都会回来找你算账”。

#### 3.2.1 状态机（建议字段）

新增 `world.xDrama`：

- `stage: null | "spark" | "callout" | "aftermath"`
- `intensity: 0-100`
- `topic`: string（本次争议主题）
- `weeksLeft: number`（剩余几周）

触发条件：

- 当推特输出结果为 `meltdown` 或 `viral` 且 `xHeat` 高时，进入 `spark/callout`。

#### 3.2.2 事件链条示例（3 选 1｜更嘴臭版本）

**链 A：被人 quote + 逐句挑刺（经典：截图+断章取义+“懂哥”出没）**
- Week 0：被 quote，节奏起飞（inbox/或弹窗）
  - 选项：删帖装死（“我没说过”） / 认错补前提（“我刚才确实嘴快了”） / 硬刚（“你不懂”）
- Week 1：截图传播（硬刚更容易上热搜），或风向缓和（认错更容易降温）
  - 选项：发澄清长文（写作地狱） / 拉黑对线（短期爽，长期埋雷） / 低调做事（“先把活干了再说”）
- Week 2：后续：品牌受损 / 或“因祸得福”（把澄清写成高质量复盘，反而涨声望）

**链 B：证据不足，被人当场“要交易哈希/要 PoC/要复现”**
- 触发：你发了“看似确定结论”的洞，但证据不够硬
  - 选项：补 PoC（爆肝但能翻盘） / 公开更正（体面止损） / 假装没看见（你以为能混过去？）
  - 后果：补 PoC 会消耗精力，但可能把翻车变成“高质量输出”；装死则可能升级为“信誉受损+被持续追着喷”

**链 C：合规/COI 打脸（最脏的那种：你以为赚到了，其实是买麻烦）**
- 触发：接了私货/咨询，被人扒出来“商单未披露/利益相关”
  - 选项：公开披露（合规+，名声不一定保得住） / 私下道歉（看人品） / 继续嘴硬（那就别怪别人把你挂墙上）
  - 后果：合规风险、公司信任、声望都可能受影响；在职时更容易触发“公司政治/劝退”链条

### 3.3 事件密度：更“常驻”，但更多放在 inbox

把事件拆成两类：

1) **强制弹窗事件**（重大抉择/硬后果）：维持“1~2 周至少 1 次”，但新增“推特剧情”作为重要来源。  
2) **可选事件（inbox）**：每周**保证**有足够的“可点内容”，并且其中一部分要能改变资源/机会（而不只是文案）。

建议数值：

- 每周生成 inbox：**5~8 条**（其中至少 2 条是 security/industry，至少 1 条是 x/社交）
- 强制弹窗事件：**平均每 1~2 周 1 次**，但当 `xHeat` 或 `xDrama` 处于活跃时，提高权重（让推特玩法更有“活着的世界”）

> 体验目标：玩家每周都能在右侧“事件列表”里看到可点内容；但不会每周都被弹窗打断节奏。

---

### 3.4 拉长游戏：赛季/时间门槛 + 里程碑胜利条件

#### 3.4.1 引入“赛季长度”设置（推荐）

新增设置：`seasonWeeks`（默认 52 周；可选 12/24/36/52）。

- **12 周**：快节奏体验（适合测试）
- **52 周**：默认正玩法（避免 3 个月就“见顶”）

#### 3.4.2 胜利条件加“时间门槛”

胜利结局增加一个最低周数门槛（避免 12 周就封神）：

- `progress.totalWeeks >= 24` 才允许触发任何 **win** 结局  
  （或更硬：`>= 36`，看体验）

#### 3.4.3 胜利条件加“里程碑”

把胜利从“纯数值阈值”升级为“数值 + 交付证明”：

示例里程碑（任取其一组合作为 win 前置）：

- **交付型**：累计交付直客/比赛总数 >= N（例如 6~10）
- **稳定型**：连续 K 周现金为正且合规<阈值（例如 8 周）
- **平台型**：平台结算次数 >= N 且 评级>=阈值（例如 5 场 + 70）
- **品牌型**：推特输出累计次数>=N 且 触发过至少 1 次“打脸→反转/澄清成功”

> 直观：玩家必须“活过一段时间、做过一些真实事”，才算真正封神。

---

### 3.5 成长系统：学习不稳定，实践更稳定（“理论 vs 实战”）

#### 3.5.1 学习（learn）改为“可能无效”

将 learn 的成长改为随机：

- 40%：小幅提升 +1~2
- 45%：无提升（但可能回一点心态/精力）
- 15%：走神/刷短视频（心态-1 或 精力-1）

目的：让玩家不再“无脑学习就变强”。

#### 3.5.2 实战成长：打比赛/真实交付带稳定提升

新增“实践成长”机制（不一定立刻做成复杂 XP，先做轻量也行）：

- **平台竞赛结算**：
  - 有提交且有通过/去重/驳回都会有“复盘成长”
  - 写作/审计/工具链获得小幅稳定提升（例如 +1~2）
- **直客交付**：
  - 报告完成度高 + 复测到位 → writing/comms/skill 小幅提升
- **推特高质量输出**：
  - ok/pop/viral 对 writing/comms 给予极小幅成长（例如 +0~1）作为“长期运营”回报

目的：让玩家感觉“打比赛确实能变强”，学习只是补充。

## 5. 验收标准（测试体验指标｜不靠玄学、靠体感爽点）

- **私货门槛生效**：低声望/低关系网时基本别想“咨询来钱”；达到门槛后明显增加（而且钱越多越可能带合规麻烦）。
- **推特失败有剧情**：至少 1 条“打脸流程”能持续 1~3 周，选择能明显影响后果（道歉真能降温、嘴硬真会升级）。
- **事件不再“空”**：每周 inbox ≥ 5 条，而且至少 1 条能改变机会/资源（带来订单/小钱/声望变化），不是纯段子。
- **不会 12 周就封神**：默认赛季 52 周；win 结局至少到 24 周后才可能触发（别再 3 个月速通人生）。
- **成长更像现实**：learn 会无效；但平台/交付/复盘能更稳定提升——靠挨打变强，别靠按钮变强。
